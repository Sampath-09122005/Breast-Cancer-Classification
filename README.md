# 🩺 Breast Cancer Classification using Feedforward Neural Network (FFNN)

A Deep Learning project that classifies breast cancer tumors as **Benign** or **Malignant** using a Feedforward Neural Network (FFNN) built with **PyTorch**.

---

## 📌 Project Overview

This project implements a Feedforward Neural Network (FFNN) with:

- **20 Input Features**
- **5 Hidden Layers**
- **10 Neurons in each Hidden Layer**
- **ReLU Activation**
- **Sigmoid Output Layer**

The model predicts whether a breast tumor is **Benign** or **Malignant**.

---

## 🛠️ Features

- Feedforward Neural Network (FFNN)
- Batch Normalization
- Dropout Regularization
- Early Stopping
- Comparison of SGD and Adam Optimizers
- Comparison of BCE and MSE Loss Functions
- ROC Curve & AUC Analysis
- Confusion Matrix
- Precision, Recall, Specificity and F1-Score
